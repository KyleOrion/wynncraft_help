# Wynncraft Helper

A Next.js website for Wynncraft MMORPG players to understand game mechanics, classes, and build optimization.

## Tech Stack

- **Next.js 16** - React framework with App Router
- **TypeScript** - Type-safe development
- **MUI Joy** - Modern Material UI component library
- **MUI Icons** - Icon set

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3005](http://localhost:3005) in your browser.

## Features

- Overview of all 5 Wynncraft classes
- Detailed skill points system explanation
- Getting started guide for new players
- Clean, mobile-responsive design with dark mode
- Built with the same tech stack as kylebethke.com

## Future Features

- Interactive build calculator

## Development

```bash
npm run build      # Build for production
npm run start      # Start production server
npm run lint       # Run ESLint
npm run type-check # Run TypeScript type checking
```
